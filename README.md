## 🚀 Publicación en GitHub Pages

1. Ir a **Settings → Pages**  
2. En **Source**, seleccionar `Deploy from a branch`.  
3. Elegir la rama `main` y la carpeta `/ (root)`.  
4. Guardar cambios.  
5. La URL pública se genera en menos de 2 minutos.
